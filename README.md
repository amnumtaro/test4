# 🔮 Тест: Ваше энергетическое состояние

Интерактивный тест для диагностики энергетического состояния по системе Матрицы Судьбы.

## ✨ Что вы получите, пройдя тест:

- ⚡ Увидите, в каком энергетическом состоянии вы находитесь прямо сейчас
- 🔮 Поймете, какие именно энергии в вашей Матрице сейчас «спят» или работают в минусе
- 💫 Получите первую подсказку, с чего начать мягкий выход из выгорания

## 🚀 Как использовать

### Способ 1: GitHub Pages (рекомендуется)

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы (`index.html`, `style.css`, `script.js`, `README.md`)
3. Перейдите в **Settings → Pages**
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`
5. Нажмите **Save**
6. Через 1-2 минуты ваш тест будет доступен по адресу: `https://ваш-username.github.io/название-репозитория`

### Способ 2: Локальный запуск

1. Скачайте все файлы
2. Откройте `index.html` в браузере
3. Готово!

## 🎯 Как работает тест

- **5 вопросов** о различных аспектах жизни
- **4 уровня результатов** от высокого до критического
- **Персональные рекомендации** для каждого уровня
- **Адаптивный дизайн** - работает на телефонах и компьютерах
